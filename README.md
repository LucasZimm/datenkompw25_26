# example implementation for exercise F4 (lossless image coding: version 1)

Building the Code:
==================

Requirements:
-------------
- c++ compiler (supporting c++11)
- cmake
- make (or Visual Studio)

Linux/Make:
-----------
`> make`

or

`> make cc=(path to c++ compiler)`

or 

`> make debug`

or

`> make cc=(path to c++ compiler) debug`

or 

`> make all`

or

`> make cc=(path to c++ compiler) all`


Windows/Visual Studio:
----------------------
`> mkdir build`

`> cd build`

`> cmake ..`

Then open created solution and build with Visual Studio
